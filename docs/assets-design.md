## ⑦ 実装時のReact参照方法

```tsx
// 静的インポート（Viteが最適化）
import bgTitle from '../assets/images/backgrounds/title/bg_title_arcana_gate.webp';

// img タグ
<img src={bgTitle} alt="" className="w-full h-full object-cover" />

// CSS background
style={{ backgroundImage: `url(${bgTitle})` }}

// 動的パス（将来的にCDN化する場合）
const CDN_BASE = import.meta.env.VITE_ASSETS_BASE_URL ?? '';
const src = `${CDN_BASE}/images/characters/units/${unitId}.webp`;
```
